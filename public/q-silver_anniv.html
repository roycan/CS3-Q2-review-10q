Silver Anniversary Lesson

In our society, being together for 25 years means a "Silver Anniversary".

Using an if..else, write a single line of JS Code to check yearTogether variable against currentYear to give a message on the console, either of these two messages, `Celebrating in n year(s)`, or `Celebrating/ted already` using templated string based on the difference between currentYear and yearTogether.

NOTE: NO curly brackets, semi-colon is needed before an else and at the very end. Use one space only between keywords, variable and operators. Remember everything is case sensitive

Example Input1:

yearTogether = 2010

currentYear = 2025

Example Output: `Celebrating in 10 year(s)`

Example Input2:

yearTogether = 2000

currentYear = 2025

Example Output: `Celebrating/ted already`

Lesson 1: Basics of if..else Statements

In JavaScript, we use the if..else statement to perform different actions based on different conditions. Here's how the syntax looks:

if (condition) {
    // code to run if the condition is true
} else {
    // code to run if the condition is false
}

For example, let's check whether a number is positive or negative:

let number = 5;
if (number > 0) {
    console.log("Positive number");
} else {
    console.log("Negative number");
}

Lesson 2: Using Templated Strings

Templated strings, or template literals, allow you to include expressions within your strings. They are enclosed by backticks (`) instead of quotes and can contain placeholders (${expression}). Here's an example:

let name = "Alice";
let message = `Hello, ${name}!`;
console.log(message); // Output: "Hello, Alice!"
