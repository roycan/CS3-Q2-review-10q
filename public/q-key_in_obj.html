Sample Inputs and Outputs

Found Jane

Lesson: Working with Object Keys

Accessing Object Keys

To check if an object contains a specific key, you can use the in operator or the hasOwnProperty method. Here's an overview:

// Using the "in" operator
if ("key" in object) {
    // Code to be executed if "key" is in the object
}

// Using the "hasOwnProperty" method
if (object.hasOwnProperty("key")) {
    // Code to be executed if the object has the key
}
