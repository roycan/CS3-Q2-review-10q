Lesson: Working with Object Properties

Checking for Properties

To check if an object contains a specific property, you can use the in operator or the hasOwnProperty method: 

// Using the "in" operator
if ("key" in object) {
    // Code to be executed if "key" is in the object
}

// Using the "hasOwnProperty" method
if (object.hasOwnProperty("key")) {
    // Code to be executed if the object has the key
}

Deleting Properties

To delete a property from an object, you can use the delete operator:

delete object.propertyName;

Combining Concepts

Combining these concepts allows you to check if a property exists in an object and delete it if it does, all in one line of code. Practice integrating conditional checks and property deletion to effectively manage object properties.

Logging Object Contents

To print the content of an object:

console.log(object);

For a more readable format, you can use JSON.stringify:

console.log(JSON.stringify(object_name));

Logging Key-Value Pairs

To log each key-value pair in an object, you can use Object.entries and forEach: 

Object.entries(object).forEach(([key, value]) => { console.log(`${key}: ${value}`); });
