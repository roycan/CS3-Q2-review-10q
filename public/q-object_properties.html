Lesson: Adding Properties to JavaScript Objects

To add a property to an existing object in JavaScript, you can use the dot notation or the bracket notation. Here's how:

Dot Notation

The dot notation is straightforward. You access the property of an object using a dot followed by the property name. Here's an example:

let object = {}; // Assume object is an empty object
    let property = 'property'; // Assume property is a string containing the property name
    
    // Adding property with an initial value of 100 using dot notation
    object.property = 100;
    
    console.log(object); // Output: { property: 100 }

Bracket Notation

The bracket notation allows you to use variables to define properties. It's particularly useful when the property names are dynamic or not valid identifiers.

let object = {}; // Assume object is an empty object
    let property = 'property'; // Assume property is a string containing the property name
    
    // Adding property with an initial value of 100 using bracket notation
    object[property] = 100;
    
    console.log(object); // Output: { property: 100 }
