for..in Loop

            
The for..in loop iterates over the properties of an object. Itâ€™s commonly used when you need to access or manipulate the properties of an object:

            
for (const key in object) {
        // Code to be executed for each property
    }

            
            
If Statement

            
The If statement allows you to perform actions based on a condition. If the condition evaluates to true, the block of code inside the if statement gets executed:

            
if (condition) {
        // Code to be executed if condition is true
    }

            
            
Template Literals

            
Template literals, also known as template strings, are a way to embed variables and expressions inside strings using backticks ` and placeholders ${/* variable */}:

            
const name = "John"; 
    const greeting = `Hello, ${name}!`; 
    console.log(greeting); // Output: Hello, John!
