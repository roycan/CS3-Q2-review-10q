JavaScript Lesson: Counting "Maybe" Responses

Using the array responses that contains "yes", "no", or "maybe", count the number of "maybe" responses and place it in totalMaybes, which has been declared and initialized to 0. Use a for..loop (not for..in) and an if statement on one line. Use ndx as the variable in the for..loop.

Sample Input and Output

Given the following inputs:

let responses = ["yes", "maybe", "no", "maybe", "yes"];
let totalMaybes = 0;

The output should be:

// totalMaybes is 2

A for loop is used to repeatedly execute a block of code a certain number of times. Here's the syntax:

for (initialization; condition; increment) {
    // code to be executed
}

For example, to print numbers from 0 to 4:

for (let i = 0; i < 5; i++) {
    console.log(i);
}

The if statement is used to perform actions based on a condition. If the condition is true, the block of code inside the if statement is executed:

if (condition) {
    // code to run if the condition is true
} else {
    // code to run if the condition is false
}

For example, to check if a number is even:

let number = 8;
if (number % 2 === 0) {
    console.log("Even");
} else {
    console.log("Odd");
}
