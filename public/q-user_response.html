Lesson: Handling User Input

To handle user input, you can use the document.getElementById method to get the input field and access its value:

let userInput = document.getElementById("yourMess").value;

Console Logging

The console.log method is used to print messages to the browser's console. It's a useful tool for debugging and testing your code:

console.log(userInput);

Combining Concepts

Combining these concepts allows you to access the user's input and log it to the console in a single line of code. Practice integrating input handling and console logging to effectively debug and test your programs.
