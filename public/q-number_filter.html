JavaScript Lesson: Number Filter

Using the array numberList that contains numbers, filter out the numbers that are divisible by both 2 and 6, and log them to the console.

Sample Input 

let numberList = [12, 18, 23, 36, 48];

Solve it with Your Code


To filter numbers in an array, you can use the filter method. Here's a weird idea as an example: 

let numberList = [12, 18, 23, 36, 48];
let filteredNumbers = numberList.filter(num => num % 12 === 0);
console.log(filteredNumbers);

Combining Both on One Line

To accomplish the task on one line, you can use both the for..in loop and the if statement in a single statement:

for (const index in numberList) if (numberList[index] % 12 === 0) console.log(numberList[index]);

Here, we use % 12 === 0 because a number divisible by both 2 and 6 is also divisible by 12 ?!?
