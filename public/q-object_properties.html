JavaScript Lesson: Object Properties

Create the property dProp inside dObj with an initial value of 100. Assume that dProp and dObj have been declared and initialized already.

Initial code 

let dObj = {}; // Assume dObj is an empty object
let dProp = 'dProp'; // Assume dProp is a string containing the property name

Solve it with Your Code


To add a property to an existing object in JavaScript, you can use the dot notation or the bracket notation. Here's how:

Dot Notation

The dot notation is straightforward. You access the property of an object using a dot followed by the property name. Here's an example:

let dObj = {}; // Assume dObj is an empty object
let dProp = 'dProp'; // Assume dProp is a string containing the property name

// Adding dProp with an initial value of 100 using dot notation
dObj.dProp = 100;

Bracket Notation

The bracket notation allows you to use variables to define properties. It's particularly useful when the property names are dynamic or not valid identifiers.

let dObj = {}; // Assume dObj is an empty object
let dProp = 'dProp'; // Assume dProp is a string containing the property name

// Adding dProp with an initial value of 100 using bracket notation
dObj[dProp] = 100;
